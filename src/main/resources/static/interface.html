  2. 用户-设置

视频相关

  • 列表/手气不错/用户上传/收藏夹

    列表 http://47.92.29.192:8082/video/videos?type=list&direction=down&start=10&pageSize=5

    手气不错 http://47.92.29.192:8082/video/videos?type=goodluck

    用户上传 http://47.92.29.192:8082/video/videos?type=upload&pageNum=1&pageSize=10&memberId=38

    收藏夹 http://47.92.29.192:8082/video/videos?type=favourite&pageNum=1&pageSize=10

    type list-视频列表 goodluck-手气不错 upload-用户上传 favourite-收藏列表

    direction up|down

    start 起始位置

    memberId 订阅的会员id

    pageNum 当前页（从1开始）-仅在type=favourite|upload的时候有效

    pageSize 页大小-仅在type=list|favourite|upload的时候有效

  • 评论列表

    分页：http://47.92.29.192:8082/video/videoComments?videoId=57&type=list&pageSize=10&pageNum=1

    5条热评：http://47.92.29.192:8082/video/videoComments?videoId=57&type=latest

    type latest-选取最热的五条（和一条当前用户的） list-分页显示

    videoId 视频id

    pageNum 当前页（从1开始）-仅在type=list的时候有效

    pageSize 页大小-仅在type=list的时候有效

  • 添加评论

    http://47.92.29.192:8082/video/addVideoComment?videoId=57&commentText=卡哇伊！

  • 操作

    http://47.92.29.192:8082/video/operation/52?type=video&operation=like

    id 视频/评论id

    type video/comment

    operation 操作：（favourite/cancelFavourite/like/unlike/accuse）
